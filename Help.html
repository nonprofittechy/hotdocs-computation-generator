The HotDocs Computation Generator simplifies the repetitive work of scripting a large number of components with HotDocs Developer, version 11 or newer. An earlier version was developed in PowerShell.

Select a Hotdocs 11+ component file that is stored on Google Drive, or upload a new component file to get started.

The tools here will help you:

  1. Load a list of components from a dialog into a spreadsheet for you to manipulate.
  2. Add text (like SHOW or HIDE) keywords before/after selected components.
  3. Create a series of IF/THEN statements to conditionally set the value of a component based on an earlier selection.
  4. Create summary paragraphs based on true/false components.
  5. Generate REPEAT arrays from a sheet (as of 2017, mostly useful for interviews hosted on LawHelpInteractive).

This add-on is non-destructive. The functions will generate text that you can copy into the HotDocs script editor but won't change the component file itself. The functions will load the most recent information from the component file in Google Drive each time that they run. For your convenience, when you select a component file, a new sheet will be created that lists all of the components in the file, which you can refresh when you like. Changes made on this sheet do not change the component file.
